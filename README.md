# Task Manager

## Описание проекта

Task Manager — это веб-приложение для управления задачами.
Пользователь может создавать задачи, редактировать их, менять статус, сортировать и фильтровать.
Также реализована **Kanban-доска**, которая позволяет перетаскивать задачи между колонками.

Проект реализован на **Next.js** с использованием **React** и **Tailwind CSS**.

---

# Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone <repository-url>
```

или просто скачать архив проекта.

### 2. Перейти в папку проекта

```bash
cd task-manager
```

### 3. Установить зависимости

```bash
npm install
```

### 4. Запустить проект

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```
http://localhost:3000
```

# Используемые технологии

В проекте использованы следующие технологии:

* **Next.js** — основной фреймворк
* **React** — построение интерфейса
* **TypeScript** — типизация кода
* **Tailwind CSS** — стилизация интерфейса
* **React Hooks** — управление состоянием
* **hello-pangea/dnd** — drag-and-drop для Kanban-доски
* **LocalStorage** — сохранение темы (dark / light)

# Реализованный функционал

В проекте реализованы следующие возможности:

### Управление задачами

* создание задач
* просмотр задачи
* редактирование задачи
* отображение даты создания
* отображение статуса и приоритета

### Фильтрация

Можно фильтровать задачи по:

* статусу
* приоритету

### Поиск

Реализован поиск задач по названию.

Для оптимизации используется **debounce**, чтобы поиск не выполнялся при каждом вводе символа.

### Сортировка

Задачи можно сортировать:

* по дате создания
* по приоритету

Также можно менять порядок:

* по возрастанию
* по убыванию

### Kanban-доска

Реализована Kanban-доска с колонками:

* New
* In Progress
* Done

Задачи можно **перетаскивать между колонками**, что автоматически меняет их статус.

# Темная и светлая тема

В приложении реализовано переключение темы:

* **Light mode**
* **Dark mode**

Тема переключается через кнопку и сохраняется в **localStorage**, поэтому после перезагрузки страницы выбранная тема сохраняется.

Tailwind используется с режимом:

```
darkMode: 'class'
```

# Архитектура проекта

Структура проекта организована по feature-подходу.

```
features/
   tasks/
      components/
      hooks/
      types/

hooks/
   useDebounce
```

Это позволяет:

* разделить логику
* сделать код более читаемым
* упростить масштабирование проекта

# Принятые решения

### Использование Next.js

Next.js был выбран потому что:

* упрощает структуру проекта
* имеет удобную систему роутинга
* хорошо работает с React

### Использование Tailwind

Tailwind CSS позволяет:

* быстрее писать стили
* использовать готовые utility-классы
* легко реализовать dark mode

### Использование Kanban

Kanban-доска делает управление задачами более наглядным и удобным.

Для реализации drag-and-drop используется библиотека:

```
hello-pangea/dnd
```
